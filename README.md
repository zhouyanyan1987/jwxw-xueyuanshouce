# 镜悟小我心理成长训练营——学员使用手册

## 项目简介

这是一个为心理成长训练营设计的学员使用手册网页，采用**淡水墨禅意风格**设计，提供优雅、平静的阅读体验。版本已升级，包含自动滚动导航、水墨背景和禅意装饰元素。

## ✨ 新版本特色功能

### 🎨 视觉升级
- **水墨禅意背景**：主背景采用中国传统水墨画，营造深远意境
- **装饰性元素**：添加禅意符号、水墨纹理、山水线条等装饰
- **动态效果**：背景视差滚动、元素悬停动画、加载动画
- **毛玻璃效果**：使用backdrop-filter创建现代毛玻璃质感

### 🔄 智能导航
- **自动滚动监听**：鼠标下滑时自动高亮对应导航项
- **触摸手势支持**：上下滑动切换内容区域
- **键盘导航**：方向键快速切换，增强无障碍访问
- **滚动进度指示器**：顶部进度条显示阅读进度

### 📱 响应式优化
- **完全响应式**：适配桌面、平板、手机全设备
- **性能优化**：使用Intersection Observer和requestAnimationFrame
- **加载优化**：图片预加载和渐进式动画效果

## 设计特色

### 视觉风格
- **淡水墨极简风**：采用烟雨蓝色系和米白色背景，融合中国传统水墨画意境
- **禅意装饰**：加入禅符号、山水线条、莲花图案等东方美学元素
- **现代质感**：毛玻璃效果、渐变色彩、柔和阴影
- **动态交互**：悬停动画、滚动视差、元素浮现效果

### 核心功能
- **智能滚动监听**：自动检测用户浏览位置，同步高亮对应导航
- **多维度导航**：点击导航、手势滑动、键盘操作、滚动自动切换
- **视觉反馈**：实时滚动进度条、导航状态指示、交互动效
- **无障碍优化**：完整ARIA标签、键盘导航支持、语义化结构

### 技术亮点
- **Intersection Observer API**：高性能元素可见性检测
- **CSS Grid & Flexbox**：现代布局技术
- **Backdrop Filter**：毛玻璃效果实现
- **RequestAnimationFrame**：流畅动画性能保障

## 使用方法

### 本地预览
1. 解压文件到本地文件夹
2. 使用现代浏览器打开 `index.html` 文件
3. 或者启动本地服务器：
   ```bash
   # 使用 Python 3
   python -m http.server 8000
   
   # 使用 Node.js
   npx serve .
   ```

### 在线部署
1. 将整个文件夹上传到Web服务器
2. 确保服务器支持静态文件托管
3. 访问对应的URL地址

### 交互操作
- **导航切换**：点击左侧导航或使用方向键
- **滚动监听**：向下滚动页面，导航自动跟随高亮
- **手势操作**：移动端上下滑动切换内容
- **进度查看**：顶部滚动条显示阅读进度

## 自定义修改

### 颜色主题
在 `styles.css` 中可以找到主要色彩变量：
```css
--primary-blue: #8FA6BF;      /* 烟雨蓝 */
--primary-light: #DDE6F0;     /* 天青 */
--background-page: #F8F9FA;   /* 米白 */
--text-primary: #212529;      /* 主要文字 */
--text-secondary: #6C757D;    /* 辅助文字 */
```

### 背景图片更换
替换 `imgs/` 文件夹中的图片：
- `zen_background_3.jpg` - 主背景（水墨画）
- `ink_texture_4.jpg` - 纹理装饰

### 动画效果调整
```css
/* 滚动监听阈值 */
rootMargin: '-40% 0px -50% 0px'

/* 动画持续时间 */
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);

/* 背景视差速度 */
scrollTop * 0.1  /* 主背景移动速度 */
scrollTop * 0.05 /* 纹理移动速度 */
```

### JavaScript功能开关
```javascript
// 启用/禁用自动滚动监听
setupScrollSpy();

// 启用/禁用视差效果
setupParallaxEffect();

// 启用/禁用触摸手势
handleTouchNavigation();
```

## 浏览器兼容性说明

### 完全支持
- Chrome 70+ ✅
- Firefox 65+ ✅
- Safari 12+ ✅
- Edge 79+ ✅

### 部分支持（降级处理）
- Internet Explorer 11 ❌（不推荐使用）
- 旧版移动浏览器 ⚠️（核心功能可用，部分效果降级）

### 降级策略
- 不支持 backdrop-filter 的浏览器会使用半透明背景
- 不支持 Intersection Observer 的浏览器会禁用自动滚动监听
- 不支持 CSS Grid 的浏览器会使用 Flexbox 布局
