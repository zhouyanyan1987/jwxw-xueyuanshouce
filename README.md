# 镜悟小我心理成长训练营 - 水墨国风终极版

## 项目简介

这是一个基于水墨国风设计的心理成长训练营学员手册，采用现代化的HTML5技术构建，具有完整的动画效果和交互体验。

## 特色功能

### 🎨 设计特色
- **水墨国风设计**：融合传统文化元素与现代设计美学
- **完整色彩系统**：21种颜色，5个色系（水墨、宣纸、朱砂红、传统金、自然绿）
- **装饰元素**：12个动态装饰元素（山水画、竹叶、墨滴、印章）
- **渐变与阴影**：复杂的CSS渐变和阴影系统

### 📱 交互功能
- **多种导航方式**：
  - 键盘导航（方向键、空格键、Home/End）
  - 鼠标滚轮导航
  - 触摸滑动导航（移动设备）
  - 进度指示器点击导航
- **动画效果**：
  - 页面切换动画（淡入淡出）
  - 内容进入动画（淡入上升、滑入右侧、传统入场）
  - 装饰元素动画（山水浮动、竹叶摇摆、墨滴旋转、印章发光）

### 📋 内容结构
1. **欢迎封面** - 温馨的欢迎词和装饰元素
2. **参营前准备** - 7个准备事项的卡片展示
3. **训练营流程（第一天）** - 重点时间安排
4. **训练营完整流程** - 三天完整时间表
5. **训练营形式** - 8种不同的学习形式
6. **如何达到最佳学习效果** - 6个关键要点
7. **参营后服务须知** - 后续服务说明
8. **结束语** - 祝福和总结

### 🔧 技术特性
- **响应式设计**：完美适配桌面端、平板端、移动端
- **性能优化**：
  - CSS变量系统，便于维护
  - 防抖优化的用户交互
  - 页面可见性检测，自动暂停/恢复动画
  - 预加载优化
- **打印友好**：专门优化的打印样式
- **无障碍支持**：键盘导航、标题提示等

## GitHub Pages 部署

### 快速部署

1. **Fork项目到您的GitHub仓库**

2. **启用GitHub Pages**：
   - 进入仓库设置
   - 找到"Pages"选项
   - Source选择"Deploy from a branch"
   - Branch选择"main"或"master"
   - Folder选择"/ (root)"

3. **访问您的网站**：
   ```
   https://your-username.github.io/repository-name
   ```

### 自定义域名（可选）

如果您有自定义域名，请在仓库根目录添加`CNAME`文件，内容为您的域名：

```
your-domain.com
```

## 开发说明

### 样式系统
项目使用CSS变量系统，所有颜色和间距都通过变量定义：

```css
:root {
    --ink-black: #1A1A1A;
    --vermillion-deep: #CC2936;
    --space-xl: 2rem;
    /* 更多变量... */
}
```

### 动画系统
支持多种动画类型：
- `.fade-in-up`：淡入上升
- `.slide-in-right`：滑入右侧  
- `.traditional-entrance`：传统入场

### 响应式断点
- 移动端：≤ 768px
- 桌面端：> 768px
