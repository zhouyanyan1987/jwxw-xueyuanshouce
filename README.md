# 镜悟小我心理成长训练营——学员使用手册

## 项目简介

这是一个为心理成长训练营设计的学员使用手册网页，采用淡水墨极简风格设计，提供优雅、平静的阅读体验。

## 文件结构

```
镜悟小我学员使用手册/
├── index.html          # 网页主文件
├── styles.css          # 样式文件
├── script.js           # JavaScript交互功能
├── README.md           # 项目说明文件
└── user_input_files/   # 原始资料（参考）
    ├── 学员手册_content.md
    └── 镜悟小我——学员手册.docx
```

## 设计特色

### 视觉风格
- **淡水墨极简风**：采用烟雨蓝色系和米白色背景
- **优雅字体**：思源宋体（标题）+ 思源黑体（正文）
- **宁静布局**：超大间距和非对称布局，营造呼吸感

### 功能特性
- **响应式设计**：适配桌面、平板、手机
- **侧边导航**：左侧内容导航，右侧具体内容
- **平滑动画**：优雅的内容切换效果
- **移动优化**：触摸手势支持，移动端菜单折叠

### 内容模块
1. **欢迎语** - 温馨的欢迎信息
2. **参营前准备** - 7项详细准备事项
3. **训练营流程** - 完整的时间安排表格
4. **训练营形式** - 8种体验形式介绍
5. **学习效果指导** - 5个关键学习要点
6. **服务须知** - 参营后服务说明

## 使用方法

### 本地预览
1. 将所有文件放在同一文件夹中
2. 使用现代浏览器打开 `index.html` 文件
3. 或者使用本地服务器：
   ```bash
   # 使用 Python 3
   python -m http.server 8000
   
   # 使用 Node.js
   npx serve .
   ```

### 在线部署
1. 将所有文件上传到Web服务器
2. 确保服务器支持静态文件托管
3. 访问对应的URL地址

## 自定义修改

### 颜色主题
在 `styles.css` 中可以找到色彩变量：
```css
--primary-blue: #8FA6BF;    /* 烟雨蓝 */
--primary-light: #DDE6F0;   /* 天青 */
--background-page: #F8F9FA; /* 米白 */
```

### 字体大小
在 `styles.css` 中可以调整字体大小：
```css
font-size: 40px;  /* H1主标题 */
font-size: 28px;  /* H2章节标题 */
font-size: 17px;  /* 正文 */
```

### 布局间距
修改间距变量来调整整体布局密度。
