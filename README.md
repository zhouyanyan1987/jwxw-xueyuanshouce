## 部署说明

### GitHub Pages 部署
1. 将所有文件上传到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择`main`分支作为源
4. 等待几分钟让网站生效

### 本地预览
```bash
# 使用Python内置服务器
python -m http.server 8000

# 使用Node.js serve
npx serve .
```

## 自定义配置

### 颜色系统
在`styles.css`中的`:root`部分可以修改主题颜色：
```css
:root {
    --primary-color: #2C3E50;      /* 主色调 */
    --accent-color: #E67E22;       /* 强调色 */
    --gray-500: #9AA0A6;           /* 中性色 */
    /* 更多颜色变量... */
}
```

### 动画时长
```css
:root {
    --duration-fast: 0.15s;        /* 快速动画 */
    --duration-normal: 0.3s;       /* 标准动画 */
    --duration-slow: 0.5s;         /* 慢速动画 */
}
```
